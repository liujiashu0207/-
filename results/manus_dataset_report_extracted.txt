.map 文件格式说明（Python读取）
.map 文件为纯ASCII文本，结构如下：

type octile
height 49
width 49
map
@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@.
...（每行为一行地图数据）

字符含义：
. 或 G → 可通行（free cell）
@ 或 O → 不可通行（obstacle）
T → 树木（不可通行）
S → 沼泽（可通行但代价高）
W → 水域（不可从陆地直接进入）

Python读取示例代码：

import numpy as np
 
def load_map(filepath):
    """
    读取MovingAI .map文件，返回二值栅格地图
    0 = 可通行, 1 = 障碍物
    """
    with open(filepath, 'r') as f:
        lines = f.readlines()
    
    # 解析头部
    header = {}
    map_start = 0
    for i, line in enumerate(lines):
        line = line.strip()
        if line == 'map':
            map_start = i + 1
            break
        parts = line.split()
        if len(parts) == 2:
            header[parts[0]] = parts[1]
    
    height = int(header.get('height', 0))
    width = int(header.get('width', 0))
    
    # 解析地图数据
    grid = np.zeros((height, width), dtype=np.uint8)
    passable = {'.', 'G', 'S'}  # 可通行字符
    
    for row_idx, line in enumerate(lines[map_start:map_start + height]):
        for col_idx, char in enumerate(line.rstrip('\n')):
            if char not in passable:
                grid[row_idx][col_idx] = 1  # 障碍物
    
    return grid, height, width
 
# 使用示例
grid, h, w = load_map('data/benchmark_maps/dao/arena.map')
print(f"地图尺寸: {h}×{w}, 障碍物比例: {grid.mean():.2%}")
