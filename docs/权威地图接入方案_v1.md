# 权威地图接入方案 v1

## 目标
用科研社区常用公开地图替代随机地图，提升可比性与说服力。

## 推荐优先级
1. **MovingAI Pathfinding Benchmarks（首选）**
   - 常用网格地图，A* / JPS 论文广泛采用
   - 格式多为 `.map`
2. **ROS/导航常用 occupancy grid 地图（补充）**
   - 可用于工程验证与可视化展示

## 你来找图时，检索关键词
- `MovingAI benchmark map download`
- `pathfinding benchmark .map`
- `grid map benchmark A* JPS`
- `robot navigation occupancy grid dataset`

## 已支持的地图格式
本项目实验脚本现在支持：
- `.map`（MovingAI）
- `.txt`（0/1网格）
- `.csv`（0/1网格）

## 运行方式（权威地图模式）
把地图文件放进同一目录（例如 `data/benchmark_maps`），然后运行：

```bash
python code/experiments/run_b_route_experiments.py --map_dir data/benchmark_maps --repeats 30 --seed 42 --include_ablation --out_prefix exp_benchmark_v1
```

可选限制地图数量：

```bash
python code/experiments/run_b_route_experiments.py --map_dir data/benchmark_maps --map_limit 10 --repeats 30 --seed 42 --include_ablation --out_prefix exp_benchmark_v1
```

## 输出文件
- `results/exp_benchmark_v1_raw_records.csv`
- `results/exp_benchmark_v1_summary.csv`
- `results/exp_benchmark_v1_sampling_trace.csv`
- `figures/runtime_comparison_exp_benchmark_v1.png`

## 审稿人视角注意事项
1. 地图来源必须在论文中写清楚（数据集名称 + 下载链接 + 下载日期）。
2. 如果文献对比使用固定起终点场景（scen），你也要同步采用。
3. 当前版本默认“同图随机起终点”，适合方法比较；若要严格复现实验，后续再接入官方 scen 文件。
