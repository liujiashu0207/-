# 消融实验执行清单 v1

## 目的
验证改进A*三个关键模块是否真实贡献性能：
1. 自适应权重
2. JPS-like加速
3. 路径平滑

## 运行命令

### 冒烟测试（已验证可运行）
```bash
python code/experiments/run_b_route_experiments.py --sizes 20,40 --ratios 0.3,0.7 --repeats 5 --seed 42 --include_ablation --out_prefix exp_ablation_smoke
```

### 正式版本（建议）
```bash
python code/experiments/run_b_route_experiments.py --sizes 20,40,80,100 --ratios 0.3,0.7 --repeats 30 --seed 42 --include_ablation --out_prefix exp_ablation_v1
```

## 输出文件
- `results/exp_ablation_v1_raw_records.csv`
- `results/exp_ablation_v1_summary.csv`
- `results/exp_ablation_v1_sampling_trace.csv`
- `figures/runtime_comparison_exp_ablation_v1.png`

## 消融组定义
- `ablation_no_adaptive`：去掉自适应权重（固定权重）
- `ablation_no_jump`：去掉JPS-like加速
- `ablation_no_smoothing`：去掉两阶段平滑

## 验收标准
1. 所有消融组 `success_rate = 1`（在可达样本条件下）
2. 至少一个主指标（时间/转弯/路径长度）相对 `improved_astar` 出现可解释退化
3. 退化方向与模块作用一致（例如去平滑后转弯次数上升）
