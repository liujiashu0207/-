## 协作机制说明

本仓库是 **Manus AI** 与 **Cursor** 的共同工作台，GitHub 是两者之间的实时桥梁：

- **Cursor（本地）**：负责代码编写、章节草稿填充、本地实验执行。
- **Manus（云端）**：负责文献调研、算法推导、数据审查、图表生成、全局逻辑审核。
- **用户**：最终决策者，在关键节点拍板方向。

每次 Manus 完成一项工作后，会将结果 `git push` 到本仓库，Cursor 可通过 `git pull` 直接获取并使用。
