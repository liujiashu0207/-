# Result Audit Stats

## 目标
把“看起来更好”转成“统计上可支持”的结论。

## 触发词
- 显著性、p 值、q 值、FDR、BH、统计报告、结论强度、可宣称

## 执行步骤
1. 确认比较对象与样本来源一致（同口径同数据源）。
2. 对核心指标执行显著性检验（按项目既定方法）。
3. 多重比较使用 BH-FDR 校正并输出 q 值。
4. 给出“可宣称/谨慎宣称/不可宣称”分级。
5. 生成结论-证据对照摘要。

## 输出模板
```markdown
## 统计审计结论
- 数据源：
- 比较对象：

| 指标 | 效应方向 | p_value | q_value | 是否显著 | 结论强度 |
|---|---|---:|---:|---|---|

## 可宣称结论
- ...

## 不可宣称结论
- ...（原因）
```

## 判定规则
- 仅当效应方向正确且 q 值达标，才标“可宣称”。
- 不以单次结果替代统计结论。
