# Paper Section Writer ZH

## 目标
把实验证据转成审稿人可接受的章节文本。

## 触发词
- 摘要、方法、实验设计、结果讨论、结论、改稿、润色、投稿版

## 写作规则
- 所有关键数值必须来自已确认结果文件。
- 不把“趋势”写成“显著提升”。
- 区分工程改进与理论保证（避免过度宣称最优性）。
- 每段尽量包含“结论 + 证据来源 + 含义”。

## 执行步骤
1. 读取对应章节草稿和最新权威结果。
2. 更新错误数值、口径冲突和术语不一致。
3. 按目标期刊风格压缩/扩展段落长度。
4. 输出“可直接替换段落”与“修改说明”。

## 输出模板
```markdown
## 章节更新
- 目标章节：
- 数据来源：

## 可直接替换文本
（正文）

## 修改说明
1. ...
2. ...

## 风险提示
- ...
```

## 质量门禁
- 发现“代码参数与文中参数不一致”时，先修一致性再润色。
- 无证据支持的句子必须降级表述或删除。
