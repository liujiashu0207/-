# Submission Packager CN

## 目标
在投稿前给出一套完整、可复核、可交付的材料包。

## 触发词
- 投稿、提交、打包、最终版、复现清单、材料检查、cover letter

## 打包内容
- 主文稿与章节版本
- 论文主图与图号映射
- 关键结果 CSV 与结论-证据对照表
- 一键复现命令
- 风险清单（未解决项）

## 执行步骤
1. 生成“投稿包清单”并逐项核对文件存在性。
2. 校验文中结论是否可追溯到 CSV/图。
3. 产出最终复现命令块（可直接运行）。
4. 输出“可投稿/暂不可投稿”判定与原因。

## 输出模板
```text
## 投稿包清单
| 类别 | 文件 | 状态 |
|---|---|---|

## 复现命令
`python code/experiments/run_fix15_v3.py`
`python code/visualize/plot_paper_figures_v3.py`

## 最终判定
- 结论：可投稿 / 暂不可投稿
- 原因：
- 最后补洞：
```

## 质量门禁
- 缺任一核心文件（主结果、主图、结论证据表）则不通过。
- 不能复现的结论不得进入最终稿。
